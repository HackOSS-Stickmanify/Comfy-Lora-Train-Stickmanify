This repo is for using comfyui-cli to run workflows and train LoRAs with and without UI.

## RunPod Setup Order

Run these `.sh` files in this exact order to set up the environment:

1. `init_setup.sh`
2. `install_npm.sh`
3. `install_comfy_deps.sh`
4. `init_ai_toolkit.sh`
5. `download_models.sh`

## Launch Commands (after setup)

Use these after the setup steps above complete:

- `launch_ai_toolkit.sh` (starts AI Toolkit UI)
- `launch_comfyui_gui.sh` (starts ComfyUI)
